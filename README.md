# jupyter_grabcut

Python Jupyter Notebook executing the OpenCV grabcut algorithm to remove background from given image, using interactive HoloViews objects/streams.

## [OpenCV](https://docs.opencv.org/3.4.3/)
* [Grabcut](https://docs.opencv.org/trunk/d8/d83/tutorial_py_grabcut.html)

## [HoloViews](http://holoviews.org/)
* [Boxedit](http://holoviews.org/reference/streams/bokeh/BoxEdit.html#bokeh-gallery-boxedit)
* [Freehanddraw](http://holoviews.org/reference/streams/bokeh/FreehandDraw.html#bokeh-gallery-freehanddraw)
